 นางสาว กนกววณ ดีละอ่อน 
61011212020  sec.1

 การนำเข้าไฟล์ข้อมูลแบบ Excel เป็นข้อมูลแบบ Shape File ใน ArcGIS 

1.เข้าไปโหลดข้อมูลพื้นฐานสถานศึกษาใน
https://data.go.th/DatasetDetail.aspx?id=8548e3ab-00bf-4eae-b29a-156a4aa52c0d&AspxAutoDetectCookieSupport=1



2.จะได้ไฟล์ข้อมูลพื้อฐานแบบ Excel จากนั้นเลือกข้อมูลสถานศึกษาเลือกจังหวัดที่ต้องการจะทำ shape Flile โดยไปที่เครื่องมือ การเรียงลำดับและการกรอง 



- แล้วไปที่ Province เลือกข้อมูลสถานศึกษาจังหวัดที่ต้องทำ Shape File  แล้วจะได้ข้อมูลสถานศึกษามาจากนั้นทำการ Save ไฟล์ Excel  ข้อมูลสถานศึกษา

-จากนั้นไปดูค่าพิกัดของแต่ละเซลล์ข้อมูลของแต่สถานศึกษา ถ้ามีค่าพิกัดผิดหรือไม่มีค่าพิกัด ให้ทำการแก้ไขค่าพิกัด  โดยไปที่ Google Earth Pro ทำการค้นหาสถานศึกษา แล้วทำการ ปักมุดเอาค่าพิกัดนั้นมาใส่ใน Excel มาจากนั้นทำการ Save ไฟล์ Excel  ข้อมูลสถานศึกษา



3.ไปที่ ArcCatalog แล้วเปิดไฟล์ Excel  ข้อมูลสถานศึกษา โดยไปที่ Connect To Folder ทำการ add 
แฟ้มงานที่เก็บไฟล์ Excel เข้ามา


-จากนั้นไปคลิกขวาที่ไฟล์ Excel ข้อมูลสถานศึกษา แล้วไปที่ Create Feature Class >> From XY Table



-Input Fileds ให้เลือก X Fileds และ Y Fileds เป็น Longitude ,Latitude แล้ว

-Output  >> เลือก ทำการ Save ตั้งชื่อไฟล์ตามต้องการ  แล้วเลือก Save as type ให้เป็น Shapefile

4.ทำการ Add ไฟล์ที่ Save  เข้ามา แล้ว ไปที่ Preview จะได้ Shapefile ข้อมูลสถาศึกษาดังภาพ
